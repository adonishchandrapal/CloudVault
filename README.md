\# ☁️ CloudVault



CloudVault is a cloud-inspired file management system built with Python and Streamlit.



It allows users to upload, organize, manage, edit, and download files through a simple web interface.



\## 🚀 Features



\* 📂 File Management



&#x20; \* Upload Files

&#x20; \* Download Files

&#x20; \* Delete Files

&#x20; \* Search Files



\* ✏️ File Editing



&#x20; \* Open Text Files

&#x20; \* Edit File Content

&#x20; \* Save Changes



\* 📁 Folder Management



&#x20; \* Create Folders

&#x20; \* Delete Folders

&#x20; \* Organize Files into Folders



\* 📊 Dashboard



&#x20; \* Total Files

&#x20; \* Total Folders

&#x20; \* Storage Usage Overview



\## 🛠️ Tech Stack

\## ▶️ Run Locally



Install dependencies:



```bash

pip install -r requirements.txt

```



Start the application:



```bash

python -m streamlit run app.py

```
