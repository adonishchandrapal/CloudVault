\# ☁️ CloudVault



CloudVault is a cloud-inspired file management system built with Python and Streamlit.



It allows users to upload, organize, manage, edit, and download files through a simple web interface.



\---



\## 🚀 Features



\### 📂 File Management



\* Upload Files

\* Download Files

\* Delete Files

\* Search Files



\### ✏️ File Editing



\* Open Text Files

\* Edit File Content

\* Save Changes



\### 📁 Folder Management



\* Create Folders

\* Delete Folders

\* Organize Files into Folders



\### 📊 Dashboard



\* Total Files

\* Total Folders

\* Storage Usage Overview

\## ▶️ Run Locally



Install dependencies:



```bash

pip install -r requirements.txt

```



Start the application:



```bash

python -m streamlit run app.py

```
